# Synaxis Studio Desktop

Electron-based desktop application for Synaxis Studio.

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Build without packaging
npm run build:dir
```

## Features

- Chat interface with streaming support
- Model selector with configurable parameters
- Conversation history management
- Settings panel with theme toggle
- System integration (menu bar, keyboard shortcuts, notifications)
- Auto-updater support

## Tech Stack

- Electron
- React + TypeScript
- Vite
- Tamagui UI
- Zustand (state management)
